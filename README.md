# Metro Store - E-Commerce Website

## Overview
Metro Store is a modern, responsive e-commerce platform built with **HTML**, **CSS**, and **PHP**. It showcases a clean, product-focused design for an online retail experience, featuring a curated selection of electronics and accessories.

## Features
- **Responsive Product Grid**: Displays products in a clean, organized layout
- **Product Catalog**: Showcases items with images, descriptions, and pricing
- **"Order Now" Call-to-Action**: Simple purchase flow for each product
- **Modern UI Design**: Clean typography, subtle animations, and a professional color scheme

## Setup Instructions
1. **Clone or download** the repository to your local machine
2. **Place files** in your web server directory (e.g., `htdocs` for XAMPP)
3. **Configure database** (if applicable) by editing `config.php` with your credentials
4. **Access the site** via `http://localhost/your-folder-name`

## File Structure (Typical)
```
/metro-store/
├── index.php          # Main product listing page
├── style.css          # All styling rules
├── config.php         # Database configuration (if used)
├── functions.php      # Helper functions
├── /assets/           # Images, icons, etc.
└── /includes/         # Header, footer, reusable components
```
